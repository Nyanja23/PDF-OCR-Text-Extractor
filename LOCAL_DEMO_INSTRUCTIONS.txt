================================================================================
    PDF OCR EXTRACTOR - LOCAL DEMO INSTRUCTIONS
================================================================================

Your app is READY to run locally. Here's how:

================================================================================
STEP 1: UPDATE .env FOR LOCAL DEMO (SQLite - no PostgreSQL needed)
================================================================================

Open the .env file and change this line:

FROM:
  DATABASE_URL=postgresql://ocr_user:jmendes@localhost/ocr_app_db

TO:
  DATABASE_URL=sqlite:///./demo.db

This uses SQLite instead of PostgreSQL - simpler for demo, no setup needed.


================================================================================
STEP 2: ACTIVATE VIRTUAL ENVIRONMENT
================================================================================

Open terminal in the project folder and run:

  cd /home/joseph/Desktop/pdf-ocr-extractor
  source venv/bin/activate


================================================================================
STEP 3: INSTALL DEPENDENCIES (if not already done)
================================================================================

  pip install -r requirements.txt


================================================================================
STEP 4: RUN THE APPLICATION
================================================================================

  python -m uvicorn main:app --reload --host 0.0.0.0 --port 8000

OR simply:

  python main.py


================================================================================
STEP 5: ACCESS THE APPLICATION
================================================================================

Open browser and go to:

  http://localhost:8000

You'll see the landing page. From there you can:
- Sign up (email verification will show code in terminal since no email config)
- Log in
- Upload PDFs or images for OCR processing
